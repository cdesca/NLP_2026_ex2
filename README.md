# NLP_2026_ex2

## Dependencies

The code requires the following Python libraries:

| Library | Purpose |
|---------|---------|
| `pandas` | Loading and manipulating dataset of names |
| `numpy` | Numerical operations and arrays |
| `torch` | Building and training neural network |
| `torch.nn.functional` | NN functions (cross-entropy loss and softmax) |
| `matplitlib` | Visualizing training loss, validation loss, and character embeddings |


### Additional Required Resources

1. **Names files**
The datasets of Catalan names (1950s and 2010s) from Idescat. Institut d'Estadística de Catalunya:
https://raw.githubusercontent.com/bethancunningham/nlp_2026/main/noms_50s.csv
https://raw.githubusercontent.com/bethancunningham/nlp_2026/main/noms_10s.csv
